HR Analytics - Employee Attrition Prediction

Project Overview

This project focuses on analyzing employee attrition and identifying the key factors influencing employee resignation. Using Python, Machine Learning, and Power BI, the project performs exploratory data analysis, builds a predictive model, and visualizes attrition trends to support HR decision-making.

Objectives

-Analyze employee attrition patterns.
-Identify factors affecting employee turnover.
-Build a machine learning model to predict attrition.
-Visualize employee insights using Power BI.
-Provide recommendations to improve employee retention.

Key Findings

-Employees working overtime are more likely to leave.
-Job satisfaction significantly impacts attrition.
-Employees with fewer years at the company show higher attrition rates.
=Work-life balance influences employee retention.
-Employee satisfaction factors are strong predictors of attrition.
-Business Recommendations
-Improve work-life balance initiatives.
-Monitor employee satisfaction regularly.
-Provide career growth opportunities.
-Reduce excessive overtime.
-Strengthen employee engagement programs.

Project Outcome

Successfully developed an employee attrition prediction model and interactive Power BI dashboard to help organizations identify attrition risks and improve employee retention strategies.
